# Scheduling Optimization

In this app we trying to find the optimal solution for Inlay problem.

## Pre Downloding

Install the following packages:

```bash
pip install ortools
pip install tkinter
```

## Download

Download from master the zip directory.

Run "run.bat" in the inner directory.
